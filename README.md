# PySpark Sales and Revenue Analytics Pipeline

This project implements a comprehensive data pipeline using **PySpark** to process, analyze, and visualize sales data efficiently.

## Features
- # Data integration using PySpark DataFrame API and Spark SQL**  
- # Efficient joins, aggregations, and filtering for large datasets  
- # Sales insights with total revenue, average order value, and total sales metrics  
- # Data saved in **partitioned CSV format** for improved query performance  
- # Optimized for performance with **Repartitioning**, and **Caching**  

---

## Tech Stack
- **Data Processing:** PySpark  
- **Database Integration:** MySQL  
- **File Formats:** CSV  
- **Notebook Environment:** Jupyter Notebook (IPYNB)
